# Delivery App

Hello, thank you for viewing my projects.

This is a quick project implementing a delivery app to learn about app development using react native. 


## What is used to make the app?

- React native
- JavaScript
- NativeWind 
- Redux Toolkit
- Sanity (for a demo backend)
- expo (for testing app on phones)
